# Study

Study & Discuss Project ✏️

## Ground Rules

- cycle : 1 week (수요일)
- 자유로운 주제 혹은 관심 주제 중 부담없는 선에서 준비
- 발표자는 한주까지 연기 가능
- 적극적 토론 지향
- 스터디 하루전 전 내용 공유하기
- 5인이상 풀리면 주변 맛집에서 저녁먹고 이후 스터디 하기
- 스터디 타임라인을 만들자
  - 저녁: 7시 ~8시
  - 스터디: 8시 ~ 10시
  - 잡담: 10시 ~ 11시
